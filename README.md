# ⚡ CYBER CLOCK (Neo-Brutalist Edition)

![Tech](https://img.shields.io/badge/Tech-HTML5_CSS3_JS-red?style=for-the-badge)
![Style](https://img.shields.io/badge/Style-Neo--Brutalist-yellow?style=for-the-badge)

**CYBER CLOCK** adalah aplikasi jam digital berbasis web yang mengusung estetika **Neo-Brutalist** dan **Cyberpunk**. Dibangun dengan optimasi tingkat tinggi pada rendering SVG dan efisiensi JavaScript menggunakan `requestAnimationFrame`.

---

## 🚀 Key Features

* **Neo-Brutalist UI:** Menggunakan *hard shadows*, border tebal, dan grid background yang terinspirasi dari gaya desain Saweria/Gumroad.
* **Performance Optimized:** Tidak menggunakan `setInterval` biasa, melainkan `requestAnimationFrame` untuk sinkronisasi refresh rate yang mulus.
* **Scalable Vector Graphics (SVG):** Angka digital dirender menggunakan segmentasi SVG murni, memastikan tampilan tajam di resolusi manapun (4K/Retina Ready).
* **Neon Glow FX:** Efek pendaran cahaya (glow) pada segmen aktif untuk kesan futuristik.
* **Fully Responsive:** Menyesuaikan ukuran layar secara otomatis tanpa pecah.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Properties & Keyframes)
- **Logic:** Vanilla JavaScript (ES6+)
- **Graphics:** SVG 2.0 with Gaussian Blur Filters
